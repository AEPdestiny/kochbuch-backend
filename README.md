# Public Rezeptbuch
Themenbeschreibung:  Wir entwickeln eine Web-App, die Rezepte für alle sichtbar macht.
Nutzer können Rezepte öffentlich erstellen, durchsuchen und als Favoriten markieren.
die Inhalte werden dauerhaft gespeichert, sodass die Lieblingsrezepte immer verfügbar bleiben.
